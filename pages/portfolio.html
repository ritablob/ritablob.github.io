PC Game Projects

Reignbreaker


Role Engine / Language Main Tasks Duration of Work
Programming Intern Unity / C# Debug UI, UI Toolkit setup, Implementation of new Abilities, Expansion of the Attack System, QA and bugfixing September 2023 - February 2024

Description

Lockpicks meet spiked iron gauntlets in this 'medieval-punk' single-player action-roguelike. Fight through hordes of lock-operated mechs and free the world from the corrupt elite. Plunder their coffers, smash the kingdom and become The Reignbreaker.
(I was part of the Studio Fizbin team for my BA Digital Games 6-month mandatory internship)


From Champagne To Murder


Role Engine / Language Main Tasks Date Completed
Game Programming Unity / C# Fungus dialogue setup, dialogue web / keyword system implementation, Gameplay July 2023

Description

From Champagne To Murder is a detective game, which uses the innovative "Dialog Web" as its central mechanic.
You play as a detective, interviewing a suspected criminal, who happens to be a very rich lady, and with the help of a dialogue web and a map, you try to solve the mystery of the murder of her husband.
(BA Digital Games 4th Semester project)


Tempo


Role Engine / Language Main Tasks Date Completed
Game Programming Unity / C# Input / Controls, UI, Gamefeel, Quality of life, Sound implementation February 2023

Description

Tempo is a 1v1 local multiplayer fighting game based on rhythm, set in a cyberpunk fighting pit of an underground club.
Fighters adapt dance styles into their moves - hits and dodges are inspired by breakdance, pole dance, capoeira and fencing.
(BA Digital Games 3rd Semester project)


One Whisp


Role Engine / Language Main Tasks Date Completed

Description

One Whisp is a point-and-click narrative game, where you play as a depressed, substance abusing skater. Try to find your phone, interact with old friends, and learn more about the things that haunt you.

Genie Escape


Role Engine / Language Main Tasks Date Completed
Game Programming Unity / C# Player Movement, Checkpoints, Platform behaviour, Rising lava logic, UI implementation February 2022

Description

Genie Escape is a 2D platformer game, where you play as a little genie trying to escape its vessel. You, as the genie, are able to change yourself into a bubble (to fly with the wind currents) or a stone (to be heavy), which creates an interesting and challenging platforming experience.
(BA Digital Games 1st Semester project)


Game Projects that are a little different

VR Robot - A Trash Tale


Role Engine / Language Main Tasks Date Completed
Game Programming, Project Manager Unity / C# Team Management, XR Origin setup, Input System, in-game UI, testing supervision with Meta Quests July 2024

Description

VR Robot - A Trash Tale is a VR game set in the future. It's your first day at The Cleaning Company, and you are tasked to clean an Artist's studio. It's a huge mess, but it's totally doable - because you have your robot companion Ro with you!
Slice up big pieces of trash, get Ro's attention and throw trash towards it for it to catch them in its belly-dustbin, and work together to make the job easier and faster :).
(BA Digital Games 6th Semester project)
